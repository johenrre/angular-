# 后台管理
### 1. 环境依赖

 - npm
 - node
 - angular-cli	

### 2. 运行项目

- 克隆项目

```sh
git clone https://github.com/johenrre/angular-backstage-management.git
```

- 安装依赖

```sh
npm install
```

- 运行

```sh
npm start    // 导航到 http://localhost:4200/
```

- build

```
ng build -c production
```
